$.fn.uuiDrag Class

Defined in: js/uuiDrag.js:8
Module: uuiDrag

实现元素的可拖动功能

Constructor

$.fn.uuiDrag

(
  • $this
  • options
)

Defined in js/uuiDrag.js:8

Parameters:

  • $this JQuery

    被拖动的元素.

  • options Object

    配置.

    • range Dom | query

      可拖动的范围，默认为document.body.

    • enable Boolean

      是否可拖动.

    • trigger Query | dom

      拖动触发元素，默认为$this，如果指定了，trigger必须是dragger的子元素或者子query.

    • dragger Query | dom

      拖动元素，默认为$this，如指定了dragger，其必须是$this的子元素或者子query，表示是$this下所有符合dragger条件的元素都可以拖动.

    • dir String

      拖动方向，默认不限定，v水平，h竖直.

    • dragEvent String

      触发拖动的事件，默认mousedown.

    • dragStart Function

      开始拖动.传递参数 (this)实例，可以通过实例.dragger获取当前被拖动的元素

    • dragMove Function

      正在拖动.

    • dragEnd Function

      拖动结束.

Example:

$('.uuiDrag').uuiDrag({enable:0})
